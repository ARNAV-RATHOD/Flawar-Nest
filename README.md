## What is Flawar Nest?

Flawar Nest (FLN) is India's first programming language with natural language syntax. No brackets. No semicolons. No complex rules. Just write what you want — in English or Hindi — and it works.

```fln
make circle red at 200 300
animate all spinning
flaai: translate to Hindi Hello World
game mario
```

## Installation

### Step 1 — Download the extension

Download `flawarnest-fln-fladiator-1.0.0.vsix` from this repo.

### Step 2 — Install in VSCode

1. Open VSCode
2. Press `Ctrl+Shift+P`
3. Type `Install from VSIX`
4. Select `flawarnest-1.0.0.vsix`
5. Reload VSCode

Done! Create any `.fln` file and start coding.

# AI

## Language Syntax

### 2D Graphics
```fln
make circle red
make square blue 100
make triangle green at 300 200
animate all spinning
animate all bouncing
animate all pulsing
```

### 3D Graphics
```fln
make cube red
make sphere orange
make cylinder blue
animate all spinning
```

### Hindi Commands
```fln
गोला बनाओ लाल
वर्ग नीला
त्रिकोण हरा
सब घुमाओ
```

### AI Commands
```fln
flaai: analyze your text here
flaai: translate to Hindi hello world
flaai: translate to English नमस्ते
flaai: summarize your long text here
flaai: sentiment this is amazing!
flaai: generate code a todo app
```

### Database
```fln
flad db create "mydb"
flad db insert "users" "name=Arnav,city=Delhi"
flad db get "users"
flad db delete "users"
flad db backup "backup.sqlite"
```

### Web Server
```fln
flad server create "app" 3000
flad api create "/users" "GET"
flad api create "/users" "POST"
flad server start
flad server stop
```

### Login System
```fln
create login page
add email field
add password field
add name field
when login success go to home
```

### Security
```fln
flad encrypt "my secret"
flad decrypt "encoded string"
flad validate "user@email.com"
flad sanitize "unsafe input"
```

### Self-Writing
```fln
write "make a login page with blue theme"
generate "todo app with database"
```
